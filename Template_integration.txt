StoreCreditPro - Template Integration Guide
---------------------------------------------

INTEGRATE CREDIT BLOCK IN SHOPPING CART
----------------------------------------------

1. edit the file app/design/frontend/default_package/your_theme/template/checkout/cart.phtml

2. copy this code:
<?php echo Mage::helper('core/mwstorecreditpro')->formShoppingCartCredits(); ?>

3. before this code:
<?php echo $this->getChildHtml('coupon') ?>

SHOW EARNED CREDIT IN TOTALS CHECKOUT CART
-----------------------------------

1. edit the file app/design/frontend/default_package/your_theme/template/checkout/cart/totals.phtml

2. copy this code:
<?php echo Mage::helper('core/mwstorecreditpro')->buyCreditShoppingCart(); ?>

3. before this code:
<?php echo $this->renderTotals(); ?>

SHOW EARNED CREDIT IN TOTALS CHECKOUT PAGE 
-----------------------------------
Display earned and redeemed points in order review total

1. edit the file app/design/frontend/default_package/your_theme/template/checkout/onepage/review/totals.phtml

2. copy this code:
<?php echo Mage::helper('core/mwstorecreditpro')->buyCreditOnepageReview(); ?>

3. after this code:
<?php $_colspan = $this->helper('tax')->displayCartBothPrices() ? 5 : 3; ?>


SHOW EARNED CREDIT IN TOTALS CREATE ORDER ADMIN
-----------------------------------
Display earned and redeemed points in order total

1. edit the file app/design/adminhtml/default_package/your_theme/template/sales/order/create/totals.phtml

2. copy this code:
 <?php echo Mage::helper('core/mwstorecreditpro')->buyCreditCreateOrderAmin(); ?>

3. before this code:
<?php echo $this->renderTotals(); ?>

copy this code:

before this code:

<form action="" id="co-payment-form">

copy this code:
;setTimeout(updateMwFormCredit, 5000);

after this code:
